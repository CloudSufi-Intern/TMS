# TMS Frontend

Ticket Management System — React + Vite frontend.

## Tech Stack

- **React** ^19.1.1
- **React DOM** ^19.1.1
- **React Router DOM** ^7.9.1
- **Axios** ^1.12.1
- **Tailwind CSS** ^3.4.17
- **Vite** ^6.2.0

## Getting Started

### Prerequisites
- Node.js >= 18.x
- npm >= 9.x

### Installation

```bash
cd tms-frontend
npm install
```

### Running the Development Server

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
